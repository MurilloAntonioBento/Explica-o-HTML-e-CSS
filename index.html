HTML e CSS

Ao contrário do que muitos pensam, HTML e CSS não são linguagens de PROGRAMAÇÃO, o HTML é uma linguagem de marcação e o CSS é uma linguagem de estilo. O HTML (Linguagem de Marcação de Hipertexto) é a linguagem básica para a programação de um site, este site inteiro foi feito em HTML, o HTML é usado para estruturar o conteúdo de uma página web, já o CSS (Folhas de estilo em Cascata) é uma linguagem de estilo para a apresentação visual do site

SEUS BENEFÍCIOS

WebSite

Com o HTML e CSS você pode criar um site completo e profissional desde o zero, podendo adicionar o que quiser a ele.

Design Responsivo

Utilizando o CSS, você pode criar um site responsivo, ou seja, que se adapta a qualquer tipo de tela, desde uma telão gigante para também um celular pequeno, até mesmo para um relógio.

Linguagens

O HTML tem função para integrar outras linguagens dentro de um site, onde em um único site você pode utilizar várias linguagens juntas, como HTML, CSS, JavaScript, Banco de Dados, PHP e etc.

A HISTÓRIA DA INTERNET

Tudo começou durante a guerra fria, com a criação da DARPA, pelos EUA, inicialmente, a darpa queria interligar 4 pontos dos EUA, com isso, criaram uma rede chamada "arpanet" que interligava 4 computadores diferentes. Depois disso, o protocolo usado nesses computadores começou a dar errado, e aí entra o Robert Kahn. Ele criou o TCP, outro protocolo, que tratava os dados de maneira diferente, quebrando-os em pacote, ele também criou um tema muito valioso, "Internetting", que foi utilizado para mudar o nome da "Arpanet" para "Internet".

Mas o TCP ainda tinha problemas, aí veio o Vint Cerf que criou um protocolo muito importante, o IP, ou Internet Protocol, e com a criação dos dois protocolos, foi criado o protocolo mais importante de todos os tempos, o TCP/IP, que permitiu um aumento gigante no número de máquinas que faziam parte da rede. No início da história da arpanet, a rede tinha só 4 pontos, e em 1977 a rede era enorme, e o TCP/IP deu conta. No momento que a rede começou a ficar caótica, o governo dos EUA não quis cuidar mais disso, e então se deu o nome "Internet". Em 2004 praticamente o mundo inteiro já tinha acesso a internet, onde a comunicação de dados são usadas por meio de satélites e cabos submarinos
